CC is programmed in Ruby Sinatra, as it is quick and easy to learn, and is good for prototyping. HTML and CSS was used for the front-end.

CC was created with the aim of completing Young Rewired State 2015, a week-long, world-wide hackathon for young coders. A YRS project must be original and must contain some form of open data.

CC was created by a team of four people. Jack Wilsdon and Niam Patel designed and wrote the back-end code, and Ellen Higginbottom designed and wrote the front-end.

CC is an open source centralised repository of medical calculations. It also provides an API for calculations to be accessed programmatically.

CC has been designed as a web application, for ease of use and to provide the capability for public use. The visual design and calculator interaction was coded using Bootstrap.

CC is designed for use in the medical profession, to improve the nature of calculations used to find a patient's disease development risk.
